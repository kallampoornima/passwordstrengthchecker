# Password Strength Checker

A simple password strength checker that shows real-time feedback as you type.

## Features
- Shows password strength: Very Weak, Weak, Good, Strong
- Color-coded progress bar (Red, Orange, Yellow, Green)
- Blocks spaces in passwords
- Toggle to show/hide password

## How to Use
1. Download the 3 files: `index.html`, `styles.css`, `script.js`
2. Put them in the same folder
3. Open `index.html` in your browser
4. Start typing a password

## Password Requirements
Your password gets stronger with:
- 8+ characters (+1 point)
- 12+ characters (+1 point)
- 16+ characters (+1 point)
- Uppercase letters (+1 point)
- Lowercase letters (+1 point)
- Numbers (+1 point)
- Special characters (+1 point)
- Multiple special characters (+1 point)

**8 points = Strong password!**
